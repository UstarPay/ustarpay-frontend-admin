# Settings/Profile 页面

## 功能概述

这个页面实现了用户个人设置功能，包括：

### 1. 个人信息管理
- 更新用户姓名
- 更新用户邮箱
- 显示当前账户状态和套餐信息

### 2. 密码设置
- 更新用户密码
- 密码强度验证（至少8位）
- 密码确认验证

### 3. 二级密码设置
- 更新二级密码
- 二级密码强度验证（至少6位）
- 二级密码确认验证

## API 接口

页面使用以下Go后端API接口：

- `GET /tenant` - 获取租户信息
- `PUT /tenant` - 更新租户信息
- `PUT /tenant/password` - 更新密码
- `PUT /tenant/secondary-password` - 更新二级密码

## 路由配置

页面路由：`/settings/profile`

## 组件特性

- 使用Ant Design组件库
- 响应式设计
- 表单验证
- 加载状态显示
- 错误处理和用户反馈
- 标签页布局

## 使用方法

1. 在导航菜单中点击"系统设置" -> "个人资料"
2. 在相应的标签页中进行信息更新
3. 填写必要信息后点击更新按钮
4. 系统会显示操作结果反馈
