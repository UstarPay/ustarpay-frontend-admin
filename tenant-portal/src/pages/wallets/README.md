# 钱包管理页面

## 热钱包管理页面 (`HotWalletListPage`)

### 功能概述

热钱包管理页面提供了完整的热钱包管理功能，包括创建、查看、编辑、删除和配置热钱包。

### 主要功能

#### 1. **统计概览**
- 总热钱包数量
- 活跃钱包数量
- 总余额（支持8位小数精度）
- 今日交易数量

#### 2. **搜索和筛选**
- **搜索**: 支持按钱包名称或描述搜索
- **币种筛选**: BTC、ETH、USDT、USDC
- **状态筛选**: 活跃、非活跃、冻结

#### 3. **钱包列表**
- **钱包名称**: 显示名称和描述
- **币种**: 带图标的币种标签
- **状态**: 颜色编码的状态标签
- **余额**: 显示余额和锁定余额
- **地址数量**: 带徽章显示
- **自动归集**: 启用/禁用状态
- **操作**: 查看、编辑、设置、删除

#### 4. **创建热钱包**
- 基本信息：名称、币种、描述
- 归集设置：自动归集、归集阈值
- 限额设置：单笔提现限额、日提现限额
- 审批设置：是否需要审批

#### 5. **编辑功能**
- 支持修改钱包配置
- 实时更新钱包信息

#### 6. **删除功能**
- 确认删除提示
- 安全删除操作

### 技术特性

#### **状态管理**
- 使用 React Query 进行数据获取和缓存
- 本地状态管理搜索参数和模态框状态

#### **API 集成**
- 集成 `walletService` 进行钱包操作
- 支持分页、搜索、筛选等高级功能

#### **用户体验**
- 响应式设计，支持移动端
- 加载状态和错误处理
- 操作确认和成功提示

#### **类型安全**
- 完整的 TypeScript 类型定义
- 使用共享类型确保数据一致性

### 路由配置

```tsx
<Route path="hot-wallets/list" element={<HotWalletListPage />} />
```

### 页面结构

```
/hot-wallets/list
├── 页面头部（标题 + 创建按钮）
├── 统计卡片（4个统计指标）
├── 搜索筛选区域
├── 钱包列表表格
└── 创建/编辑模态框
```

### 数据流

1. **页面加载** → 获取钱包统计和列表数据
2. **搜索筛选** → 更新查询参数，重新获取数据
3. **创建钱包** → 调用API，成功后刷新列表
4. **编辑钱包** → 打开编辑模态框，提交后更新
5. **删除钱包** → 确认后调用删除API

### 依赖服务

- `walletService.getWallets()` - 获取钱包列表
- `walletService.getTenantWalletOverview()` - 获取钱包统计
- `walletService.createWallet()` - 创建钱包
- `walletService.deleteWallet()` - 删除钱包
